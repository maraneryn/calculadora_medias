# Sistema de Médias Escolares

## Descrição
Sistema desenvolvido para cálculo de médias de alunos com base em notas inseridas pelo usuário.

## Funcionalidades
- Cálculo de média aritmética
- Entrada de notas
- Validação de dados

## Tecnologias
- HTML

## Objetivo
Projeto desenvolvido para prática de lógica de programação e estruturação de sistemas.
